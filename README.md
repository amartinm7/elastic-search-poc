# Elastic-search-poc

# The Goal



Install gradle wrapper
```
gradle wrapper
```

Create scaffolding project
```bash
spring init --build gradle --name=elastic-search-poc --dependencies=web,data-jpa,mysql,devtools,thymeleaf --package-name=com.amm.artifact elastic-search-poc
```
